ACTIVIDAD UF4 ENTORNO CLIENTE

Modificar información de un archivo HTML por el contenido de un archivo *.txt externo mediante AJAX

AJAX significa JavaScript asíncrono y XML (Asynchronous JavaScript and XML). Es un conjunto de técnicas de desarrollo web que permiten que las aplicaciones web funcionen de forma asíncrona, procesando cualquier solicitud al servidor en segundo plano.Este método no permite crear varias peticiones y mantenerlas todas a la escucha para imprimir diferentes archivos externos.
Si se quieren crear varias peticiones, hay que crear varios objetos XMLHttpRequest.

El texto externo se insertará en el recuadro inferior
El scroll en el texto se realiza con el ratón, se ha ocultado la barra de scroll con CSS

Este texto se eliminará y será sustituido por el nuevo texto desde el archivo externo o al clicar en el botón ocultar texto
